# ML Concepts - Code Examples

This folder contains code examples and theory for various ML concepts.

## Linear Regression

### Files
- `01_linear_regression.md` - Comprehensive theory and concepts
- `01_linear_regression.py` - Full Python implementation
- `01_linear_regression.ipynb` - Jupyter notebook version

### Sub-topics Covered

| Section | Description |
|---------|-------------|
| 1. Simple LR | One feature, one target |
| 2. Multiple LR | Multiple features |
| 3. Gradient Descent | Optimization from scratch |
| 4. Normal Equation | Analytical solution |
| 5. Ridge (L2) | L2 regularization |
| 6. Lasso (L1) | L1 regularization + feature selection |
| 7. Elastic Net | L1 + L2 combined |
| 8. Polynomial | Non-linear relationships |
| 9. Diagnostics | Residual analysis |
| 10. Cross-Validation | K-fold CV |
| 11. Diabetes Example | Real-world dataset |
